IVAN - International Variable Administration over the Network
---------------------------

IVAN is a variable administration system designed to manage and control various parameters across a network. It allows for real-time adjustments and monitoring of variables for stateful systems. Think of it as a centralized fast database system.

Some of the key features include:
- Full CRUD operations on variables 
- distributed system integration
- consistency and data integrity

Architecture
The system is built on a client-server model, where the server manages the variables and clients can connect to read or modify them. The server uses a distributed architecture to ensure high availability and scalability.

The community implementation of IVAN is ivand, it consists of:
- ivand: TCP daemon/server that manages variables and operations described in the protocol
- ivanctl: Command-line client for interaction with a IVAN server.

The IVAN SDK will be released in the future, it will provide libraries for various programming languages to facilitate integration with IVAN servers.

Download/Compiling
You can compile ivand and ivanctl from the source code available on our GitHub repository. To compile, simply run `make`, you can then use the compiled binaries.
